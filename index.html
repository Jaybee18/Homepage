Progress

Dieses Projekt ist ein Python module, welches man bei sich im Script importieren und benutzen kann. Es soll dabei helfen Fortschritte in, zum Beispiel, grossen Datentransferprozessen zu visualisieren. Das module beinhaltet eine Klasse 'progressBar' und eine 'animation' (naehere Erklaerung auf GitHub). Die Klassen sind extra so gestaltet, dass sie eine moeglichst grosse Anzahl an verstellbaren Parametern haben, um jede progressBar so individuell wie moeglich gestalten zu koennen. Die Benutzung ist dabei auch so einfach wie moeglich gehalten, man kann jedoch auch mit verschiedenen eingebauten Methoden die progressBar manuell bedienen.

Translator

Dieses Projekt kommt in Form einer .exe Datei, wurde aber in Python geschrieben. Wenn die Datei ausgefuehrt wird, ploppt ein Pythonterminal auf. Dieses kann Minimiert werden, da es für die Benutzung der App nicht relevant ist. Das Programm wird dann, bis das Terminal wieder geschlossen wird, jedes Wort, an welches !tl[sprache] gehaengt wird, automatisch in die gewuenschte Sprache uebersetzen und das zuvor geschriebene Wort mit dem neuen Wort ersetzen. Dieses Projekt ist fuer Leute, die nicht immer den Uebersetzer aufrufen wollen muessen, wenn sie gerade im 'Schreibflow' sind und ein Wort in einer anderen Sprache nicht wissen.
